# 🧠 AI Fake News Detection System

An AI-powered web application that detects whether news is REAL or FAKE using Machine Learning + Real-time Evidence Verification.

---

## 🚀 Features

- 📰 News detection from Text / URL / Image  
- 🧠 Machine Learning based classification  
- 🔎 Google News real-time verification  
- 📊 Evidence-based confidence scoring  
- 🧠 ChatGPT-style explanation engine  
- 🔗 Clickable news source links  
- 📈 Visual confidence graph  

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- Scikit-learn  
- Newspaper3k  
- Plotly  
- BeautifulSoup  
- NLP (TextBlob, LangDetect)  

---

```bash
pip install -r requirements.txt
